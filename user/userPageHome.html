A NeoPets Shop

Quem Somos Nós

A NeoPets Shop iniciou suas atividades com a inauguração da primeira unidade em 15 de Novembro de 1999, uma loja de produtos destinados aos animais de estimação de todos os tipos. Atualmente a loja trabalha virtualmente com serviços ficticios com o intuito de criar um site para a disciplina de desenvolvimento web, portanto serão utilizados HTML e CSS para criar a fase atual do site.
